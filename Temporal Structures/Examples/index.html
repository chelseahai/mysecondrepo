Digital Object 2: Temporal Structures

This section explores temporal data visualization using D3.js, demonstrating how to create interactive timelines and event-based visualizations that help understand temporal patterns and relationships in data.

D3.js Foundation: Basic Setup and Axes

A foundational D3.js example demonstrating the core concepts needed to get started with data visualization. This example shows how to set up an SVG container, create scales to map data to pixels, and draw basic axes. It covers the essential building blocks that all D3.js visualizations are built upon.

D3.js Foundation: Timeline with Events

A foundational D3.js example demonstrating core concepts including scales, axes, data binding, and event handling. This timeline shows project milestones with interactive tooltips and color-coded categories.

Gantt Chart: Concurrent Events and Project Timeline

A Gantt chart visualization showing project phases with start and end dates, demonstrating how to visualize concurrent activities and project dependencies. Features include progress tracking, interactive tooltips, and a current date indicator.

Historic Timeline: Loading Data from CSV File

A historic timeline visualization that demonstrates how to load temporal data from external CSV files using D3.js. This example shows the process of parsing CSV data, converting date strings to Date objects, and creating visualizations from external data sources. Includes error handling for file loading issues.

Important: To run this example properly, you need to use a local server due to browser security restrictions when loading CSV files. We recommend using the VS Code Live Server extension:
  1. Install the "Live Server" extension in VS Code
  2. Right-click on this HTML file and select "Open with Live Server"
  3. This will open the page in your browser with a local server running
Modifying the CSV Data: You can customize this timeline by editing the events.csv file. The CSV structure is:
  • name: The name of the event/movement (e.g., "Renaissance", "Industrial Revolution")
  • start: The start year (e.g., 1400, 1750)
  • end: The end year (e.g., 1600, 1850)
  • category: The category for color coding (e.g., "Early Period", "Modern", "Contemporary")

Try adding your own historical events, changing the time periods, or creating different categories. The visualization will automatically update when you refresh the page!

Calendar Heatmap: Daily Data Visualization

A calendar heatmap visualization inspired by GitHub's contribution graph, demonstrating how to create a year-view calendar with daily data points. This example shows how to use D3.js to create a grid-based layout where each cell represents a day, with color intensity indicating data values.
